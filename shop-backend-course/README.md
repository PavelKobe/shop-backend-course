# Backend-курс: интернет-магазин на FastAPI

Практический курс из 16 модулей. Слой за слоем ты собираешь рабочий backend
интернет-магазина: каталог, корзина, заказы, JWT-авторизация, админка,
кэш, фоновые задачи и мониторинг — и в финале запускаешь всё одной командой
в Docker.

Стек: **FastAPI · PostgreSQL · SQLAlchemy 2.0 (async) · Alembic · JWT ·
SQLAdmin · Jinja2 · Redis · Celery · RabbitMQ · Pytest · Ruff/Mypy · Nginx ·
Prometheus · Docker**.

Каждый урок построен по схеме **Теория → Методология → Практика**.

---

## Как пользоваться (твой сценарий: GitHub + VS Code на другом ПК)

### 1. Запушить курс к себе на GitHub (с этого ПК)

```bash
cd shop-backend-course
git init
git add .
git commit -m "Backend course: интернет-магазин на FastAPI"
git branch -M main
# создай пустой репозиторий на github.com (без README), затем:
git remote add origin https://github.com/<твой-логин>/shop-backend-course.git
git push -u origin main
```

### 2. На другом ПК — склонировать и открыть в VS Code

```bash
git clone https://github.com/<твой-логин>/shop-backend-course.git
cd shop-backend-course
code .
```

В VS Code установи расширения (рекомендуется): **Python**, **Pylance**,
**Ruff**, **Even Better TOML**. Markdown-уроки открывай через
`Ctrl/Cmd + Shift + V` — будет красивый предпросмотр.

### 3. Поднять стартовый код (результат модулей M00–M01)

```bash
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\Activate.ps1
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Открой:
- http://127.0.0.1:8000/docs — интерактивная документация (Swagger UI)
- http://127.0.0.1:8000/api/v1/products — каталог-заглушка

### 4. Интерактивный хаб курса

Открой `hub/index.html` в браузере — это визуальная карта курса с отметками
прогресса (можно открывать уроки и отмечать пройденное).

## Статус контента

Все 16 модулей расписаны **полностью**: теория, методология и практика с рабочим
кодом. Код связный — проект растёт от модуля к модулю в единый интернет-магазин,
который в финале (M15) запускается одной командой `docker compose up`.
